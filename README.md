# 🌲 Words Wonderland ✨

Welcome to **Words Wonderland**, a beautifully crafted, magical typing game designed to help you improve your typing speed and accuracy while journeying through a vibrant, enchanted world. 

## 🎮 Play the Game
- **Live Web Version:** [Play on Vercel](https://words-wonderland.vercel.app/)
- **Downloadable PC Version:** [Download on Itch.io](https://praveenleo.itch.io/words-wonderland)

## ✨ Features
- **Two Unique Play Styles:** 
  - **Words Play:** Quick, satisfying bursts of typing.
  - **Sentences Play:** Test your endurance and rhythm.
- **Three Magical Realms (Difficulties):**
  - 🌱 **Easy Grove:** Perfect for beginners and casual warm-ups.
  - 🍂 **Medium Woods:** A steady challenge to test your growing skills.
  - ⛰️ **Hard Mountains:** For the ultimate typing masters!
- **Modern UI/UX:** A clean, responsive interface featuring a custom **glassmorphism** design, dynamically scaling to fit any web browser perfectly.

## 🛠️ Tech Stack
- **Game Engine:** Unity (C#)
- **UI Design:** Unity UI Canvas with fully responsive WebGL scaling.
- **Export & Deployment:** WebGL for web browsers, Vercel for hosting, and a standalone Windows `.exe` build.
